## alt.nextTick

```js
alt.nextTick(function function)
```

Executes code on next tick

#### Parameters
| Parameter Name | Type | Description |
| -------------- | ----------- | ----------- |
| function | `function` | A function to be executed on next tick |

## Example

```js
alt.nextTick(function function)
```
