## alt.clearTimeout

```js
alt.clearTimeout(int timeoutID)
```

Cancels a timeout previously established by calling setTimeout (MDN)

#### Parameters
| Parameter Name | Type | Description |
| -------------- | ----------- | ----------- |
| timeoutID | `int` | The identifier of the timeout you want to cancel |

## Example

```js
alt.clearTimeout(int timeoutID)
```
