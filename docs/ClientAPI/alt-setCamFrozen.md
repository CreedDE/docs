## alt.setCamFrozen

```js
alt.setCamFrozen(bool state)
```

Toggles camera freeze state

#### Parameters
| Parameter Name | Type | Description |
| -------------- | ----------- | ----------- |
| state | `bool` | `true` for freeze camera, `false` to unfreeze camera |

## Example

```js
alt.setCamFrozen(bool state)
```
