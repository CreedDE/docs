## alt.showCursor

```js
alt.showCursor(bool state)
```

Toggles a cursor visibility

#### Parameters
| Parameter Name | Type | Description |
| -------------- | ----------- | ----------- |
| state | `bool` | `true` for show cursor, `false` to hide cursor |

## Example

```js
alt.showCursor(bool state)
```
