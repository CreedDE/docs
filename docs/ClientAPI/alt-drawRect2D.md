```js
alt.drawRect2D(x, y, width, height, color)
```

Renders 2D Rectangle on screen

#### Parameters
| Parameter Name | Type | Description |
| -------------- | ----------- | ----------- |
| x | `int` | X Position |
| y | `int` | Y Position |
| width | `int` | Width of rectangle |
| height | `int` | Height of rectangle |
| color | `int` | Color of rectangle |
